## Project Overview
A small **Digital Piano** built using **HTML, CSS, and JavaScript** that plays musical notes when the user clicks on piano keys or presses mapped keyboard keys.

This project demonstrates the use of the **Web Audio API** (AudioContext, OscillatorNode, GainNode), keyboard and mouse events, simple ADSR-style envelopes, and a responsive user interface.  
No external audio files are used â€” all sounds are synthesized directly in the browser.

## Key Features
- Clickable piano keys (one octave + optional extra C)
- Keyboard-to-key mapping (a, w, s, e, d, etc.)
- Synthesized sound using Web Audio API
- Toggle waveform (sine / saw / triangle)
- Master volume control
- Octave up and octave down controls
- Visual active key highlight while playing
- Sustain-like behavior while holding keys
- Responsive layout for desktop and mobile devices

## Technologies Used
- HTML
- CSS
- JavaScript
- Web Audio API

## How to Use
1. Open the live demo link
2. Click on the piano keys using your mouse  
   **or**
3. Press the mapped keyboard keys to play notes
4. Adjust waveform, volume, or octave as needed
